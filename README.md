### 🏆 Projetos em Destaque

Aqui estão alguns dos desenvolvimentos recentes onde estou aplicando arquitetura e boas práticas:

- 🎬 **[Video Management API](https://github.com/Bernardocfguimaraes/API-CRUD-em-Node)**
  *Uma API RESTful desenvolvida do zero para gerenciamento de vídeos. Este projeto foi criado com o objetivo de aprofundar conhecimentos práticos em back-end, Node.js e bancos de dados relacionais estruturados na nuvem. (Node.js e SQL)*.

- 🐾 **[EloPet](https://github.com/Bernardocfguimaraes/EloPet)**
  *O EloPet é uma aplicação Fullstack desenvolvida para facilitar a criação de perfis e a arrecadação de doações para pets em situação de vulnerabilidade. O projeto foi construído com as tecnologias mais modernas do ecossistema React/Next.js, com um forte foco na arquitetura do Back-End, segurança e integridade de dados. (Next.js; Zod; Typescript)*.

- 🎮 **[HolyGames](https://github.com/Bernardocfguimaraes/holygames )**
  *O HolyGames é uma aplicação web desenvolvida para explorar, pesquisar e visualizar detalhes de jogos. Diferente de catálogos comuns, o foco deste projeto foi criar uma experiência de usuário (UX) premium e imersiva. (Next.js; TypeScript; FrameMotion).
